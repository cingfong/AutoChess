# auto chess

自走棋遊戲

使用 HTML CSS Javascript 製作

玩家可以購買棋子、布陣，棋子之間有屬性相剋，也有特殊技能處理(如補血，攻擊加成等)

目前只有單機關卡模式

## [DEMO](https://auto-chess.vercel.app/)

本地運行

```bash
npm i
npm run dev
```
